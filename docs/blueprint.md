# **App Name**: InclusiveRead Hub

## Core Features:

- Extension Overview: Introduce InclusiveRead's features, benefits, and goal of inclusive browsing. Showcase its capabilities, accessibility functions, and customisation choices, with an overview of settings to enhance content consumption.
- Installation Guide: Step-by-step tutorial that simplifies the setup process and walks users through installation for common browsers. This includes screenshots/videos showing extension store navigation and activation to make accessibility solutions easy to access for all.
- Feature Explanation: Individual guide covering functionalities like font customization, text-to-speech settings, colour adjustments, and reader mode activation; demonstrate customizing functions via screen captures and interactive demos for optimized user experience.
- Troubleshooting Section: Dedicated resource with a comprehensive FAQ and problem-solving guidance addresses common installation problems or function malfunctions; provide advice such as browser troubleshooting steps or direct extension resetting to provide ongoing accessibility.

## Style Guidelines:

- Primary color: Soft blue (#77B5FE), reflecting trust and inclusivity.
- Background color: Very light blue (#F0F8FF), creating a calm and readable environment.
- Accent color: Muted orange (#F0A34F), highlighting calls to action and important information without being distracting.
- Body text: 'PT Sans', a humanist sans-serif suitable for clear, accessible reading. Headline text: 'PT Sans'.
- Simple, clear icons to represent different functions and sections, enhancing user navigation.
- Clean, uncluttered design with plenty of white space, focusing on readability and ease of navigation.
- Subtle transitions and animations to guide users through tutorials and highlight interactive elements.
